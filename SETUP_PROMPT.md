# Project Setup Prompt

Use the following prompt if you want an AI assistant to set up this project for you, or simply follow the instructions below manually.

## 🤖 AI Prompt

Copy and paste this into your AI coding tool:

> "I have just cloned this repository. Please run the included `setup.js` script to automatically install all dependencies, configure the environment variables for both client and server, and set up the MongoDB connection. Once the setup is complete, please start the development server using `npm run dev`."

## 💻 Manual Instructions

1.  **Open a terminal** in the project root.
2.  **Run the setup script**:
    ```bash
    node setup.js
    ```
3.  **Follow the on-screen prompts** to enter your MongoDB URI.
4.  **Start the app**:
    ```bash
    npm run dev
    ```
